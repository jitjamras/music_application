# Music Application

A new Flutter project.

## Getting Started

### How to run this repo
1. Clone this repo.<br/>
2. Open file main.dart in folder Lib.<br/>
3. run command -> flutter run .<br/>
(make sure open emulator or connect mobile device)<br/>
